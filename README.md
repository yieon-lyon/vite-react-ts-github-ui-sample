## Getting Started
- dev test
    ```agsl
    yarn
    yarn dev
    ```
- production test
    ```agsl
    yarn
    yarn build
    yarn preview 
    ```
- production docker build
    ```agsl
    docker build -t vite-react-ts-github-ui-sample .
    ```
